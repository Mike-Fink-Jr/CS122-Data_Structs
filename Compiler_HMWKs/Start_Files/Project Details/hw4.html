PROBLEM DESCRIPTION

Very few arithmetic expressions in a program consist of constant numbers as their only operands. Most of them instead include variables that have been assigned in earlier statements. This assignment will therefore extend the earlier assignments by introducing variables as operands, and the assignment operator to assign to them.

The major part of this assignment may either build from Homework II or Homework III, at the student's choice. However, it is recommended that it still make use of the List Iterator class defined in Homework III.

THE ASSIGNMENT OPERATOR

Most people certainly recognize the assignment operator as one that performs an assignment operation (giving a value to a variable). But in the languages of C, Java, C++, etc. it also 'computes' a value, which can be used in a larger expression.

For example, this expression: 

	A = 3 + (B = 2)
Not only assigns 2 to B, but it adds that 2 to 3 to get a value of 5 for A.

Your program must accept this as a valid input. And of course, not all valid inputs will necessarily assign to variables, and (from Homework II), some may even build on results of earlier statements. For example, this would be a valid input line after the previous example: 

	* B
yielding a value of 10.
